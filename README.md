# 🍅 Modern Pomodoro Timer

A beautiful, high-performance Pomodoro Timer web application designed to boost productivity with a premium glassmorphic interface. Built entirely with **vanilla HTML, CSS, and JavaScript**—no frameworks or dependencies required.

## ✨ Features

- **⏱ Flexible Timer Modes**:
  - **Pomodoro** (25 min) for deep focus.
  - **Short Break** (5 min) for quick refreshes.
  - **Long Break** (15 min) for extended rest.
- **🎨 Premium Design**:
  - Sleek glassmorphism UI with smooth animations.
  - **Dark & Light Mode** support with instant toggle.
  - Animated SVG circular progress ring.
- **🔔 Smart Notifications**:
  - Chord-based audio alerts using Web Audio API.
  - Desktop browser notifications when sessions end.
  - Dynamic browser tab title with live countdown.
- **📊 Productivity Stats**:
  - Daily session counter.
  - Total focus time tracking.
  - Streak counting.
- **💾 Persistence**:
  - All settings (durations, auto-start, sounds) and stats are saved to `localStorage`.
- **⌨️ Keyboard Shortcuts**:
  - `Space`: Start / Pause
  - `R`: Reset timer
  - `S`: Skip to next session

## 🚀 How to Run

No installation is needed! 

1. Download or clone this repository.
2. Open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge).

## 📂 Project Structure

- `index.html`: The semantic structure and UI layout.
- `style.css`: The modern styling system, including theme tokens and animations.
- `script.js`: Core timer logic, session management, and persistence layer.

## ⚙️ Customization

Open the **Settings** (gear icon) to customize:
- Timer durations for all three modes.
- Auto-starting of breaks or next pomodoros.
- Sound and notification preferences.
